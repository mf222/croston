Instrucciones de Uso

Subir dos archivos con la información necesaria para realizar el análisis. El primer archivo debe contener la información sobre el consumo por SKU, y el segundo información de los parametros de reposición y leadtimes. Los archivos deben tener la siguiente estructura:

Demanda SKU

Leadtime + info

Procesar análisis

Seleccione los archivos necesarios presionando [Browse...]. Para seleccionar multiples archivos usar CTRL + click sobre cada archivo (COMMAND + click si se usa MAC). Una vez seleccionados presionar [Enviar]. Debera esperar hasta que se retorne un archivo llamado output.xlsx con el resultado. Esto podría tomar varios minutos dependiendo del tamaño de los archivos ingresados.
